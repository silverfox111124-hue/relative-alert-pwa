# 相対アラート PWA (v0.1)

Android Chrome 向けの軽量 PWA 試作品です。サーバー不要・ログイン不要・外部ライブラリなしで動作します。

## v0.1 機能

- 基準時刻入力 (`HH:mm`。例: `08:20`, `9:05`)
- プリセット選択
- `offsetMinutes` による相対時刻計算（マイナス/0/プラス対応）
- 計算結果プレビュー（時刻順）
- 通知許可リクエスト
- 通知テスト（10秒/20秒/30秒）
- セット中アラート一覧表示
- すべてキャンセル
- `localStorage` 保存（基準時刻/プリセット/セット中テスト通知）
- PWA 基本構成（manifest + service worker）

## 初期プリセット

- 外出
- 余裕あり外出
- 作業開始
- 就寝

## 使い方

1. `index.html` を Android Chrome で開く（ローカルサーバー推奨）。
2. 基準時刻を入力し、プリセットを選択。
3. 「計算してプレビュー」で一覧確認。
4. 「通知許可を確認」で通知を許可。
5. 10/20/30秒テスト通知で実機確認。
6. 必要に応じて「すべてキャンセル」。

## 動作確認方法

### PC (簡易)

```bash
python3 -m http.server 8080
```

`http://localhost:8080` を開いて UI と計算を確認。

### Android Chrome

1. 同一ネットワーク上の端末からページを開く。
2. Chrome メニューから「ホーム画面に追加」。
3. 追加後に起動し、通知テストを実行。
4. オフラインで再起動し、基本画面が開くことを確認。

## 通知の制約（重要）

- 通知は **Android / Chrome / 省電力設定 / バックグラウンド制限** に左右されます。
- `setTimeout` を使う長時間後通知は不安定です。
- v0.1 は短時間（10/20/30秒）の実機テストを最優先にしています。
- ネイティブアプリのアラームのような厳密保証はありません。
